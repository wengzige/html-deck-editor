# 帆页 Anchor Deck 品牌规范

本目录是项目品牌信息与 Logo 资产的唯一标准。README、产品界面、发布说明和宣传内容都应以这里为准。

## 固定品牌信息

| 项目 | 唯一标准 |
| --- | --- |
| 中文名 | **帆页** |
| 英文名 | **Anchor Deck** |
| 标准组合 | **帆页 Anchor Deck** |
| 一句话描述 | **让每一页，重新可编辑。** |

机器可读版本见 [`brand.json`](brand.json)。需要在代码、脚本或模板中复用品牌信息时，优先读取该文件，不要另写一套名称或文案。

## Logo

| 用途 | 文件 |
| --- | --- |
| 矢量主文件 | [`anchor-deck-icon.svg`](anchor-deck-icon.svg) |
| 1024 × 1024 位图 | [`anchor-deck-icon-1024.png`](anchor-deck-icon-1024.png) |

图标由演示页面与锚点组成。深松绿色代表稳定、专注，暖象牙白降低工具感，让整体更安静耐看。

### 使用规则

- 应用图标、头像和正方形场景只使用图标本身。
- 中文场景首次出现时使用“帆页 Anchor Deck”，后文可简称“帆页”。
- 英文场景使用“Anchor Deck”。
- 产品分支写作“帆页 Anchor Deck MCP”和“帆页 Anchor Deck 在线版”。
- 文件名、命令名和软件包名继续使用兼容性标识 `anchor-deck`，不因品牌展示名变化而改动。
- 不拉伸、旋转、添加阴影或重新绘制 Logo；不使用测试稿和历史方案。
- 不再使用其他候选中文名。

## 标准视觉

| 角色 | 色值 | 用途 |
| --- | --- | --- |
| 深松绿 | `#17624A` | Logo 主色、重点操作、品牌识别 |
| 暖象牙白 | `#F3F0E7` | Logo 线条、柔和背景 |
| 墨色 | `#17251F` | 标题和正文 |

中文优先使用苹方或系统无衬线字体，英文优先使用 Inter 或系统无衬线字体。品牌组合中“帆页”作为主名称，字重 600；“Anchor Deck”作为英文名称，字重 500。界面中保持自然字距，不使用全大写中文或装饰字体。

## 标准展示

```text
帆页 Anchor Deck
让每一页，重新可编辑。
```

除正式英文页面外，不单独用英文名替代中文品牌，也不为不同页面重新创作一句话描述。
